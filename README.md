# model_build_B

**设计:**

喂入时间输出图片. 手动对于图片进行解释分析. 撰写论文.

**使用:**

新建data目录将图片数据导入/data/Z1等.

运行backward.py训练模型.

运行test.py测试模型.

运行app.py生成结果.

**训练模型:**

图片的序号最为时间戳喂入模型(作为模型输入数据)
读图->转数组->变形喂入模型(作为模型输出数据)

**预测:**

模型预测->数组->变形->图->手动分析
